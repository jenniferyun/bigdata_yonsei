# bigdata_yonsei
데이터청년캠퍼스 연세대학교 과정 (빅데이터 분석처리) 공부기록용으로 올리는 코드입니다.

해당 과정에서는 
1) SQL을 통한 색인과정, 조인
2) 판다스를 통한 데이터시각화 
3) sklearn을 통한 회귀분석
4) 인공신경망 (CNN, GAN) 
5) 빅데이터 실무 프로젝트
등을 학습하였습니다.

해당 과정은 연세대학교 컴퓨터과학과 이원석 교수님 지도 아래 이루어졌으며, 데이터산업진흥원에서 지원하는 과정입니다. 

해당 과정의 운영 기간은 2022/06/27 - 2022/08/31 에 이루어졌습니다.
